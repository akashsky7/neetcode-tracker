# NeetCode BFSI Tracker — Akash

Your personal DSA prep tracker for JPMorgan / BNY Mellon interview prep.

## Deploy in 5 minutes (free)

### Step 1 — Push to GitHub
1. Go to https://github.com/new
2. Create a new repo called `neetcode-tracker` (keep it private if you want)
3. Upload all these files OR run:
```bash
git init
git add .
git commit -m "init tracker"
git remote add origin https://github.com/YOUR_USERNAME/neetcode-tracker.git
git push -u origin main
```

### Step 2 — Deploy on Vercel
1. Go to https://vercel.com and sign in with GitHub
2. Click **"Add New Project"**
3. Select your `neetcode-tracker` repo
4. Framework: **Create React App** (auto-detected)
5. Click **Deploy**

Done! You'll get a URL like `neetcode-tracker.vercel.app` in ~60 seconds.

### Step 3 — Bookmark it
Open the URL on your phone and desktop. Add to home screen on mobile for one-tap access.

## Notes
- Progress saves to your browser's localStorage on each device
- Free forever on Vercel (no credit card needed)
- The schedule auto-highlights today's date and opens your current week
